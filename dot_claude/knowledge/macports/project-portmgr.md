# 7.7. The PortMgr Team

The MacPorts PortMgr team is the steering group for The MacPorts Project. Its membership is typically determined by public elections among project members. The current team members can be found on the [MacPorts Developers wiki page](https://trac.macports.org/wiki/MacPortsDevelopers).

## Responsibilities

Their key responsibilities include:

- Approving new project members (granting commit rights)
- Setting general project guidelines
- Resolving disputes
- Managing project infrastructure
- Engineering releases
