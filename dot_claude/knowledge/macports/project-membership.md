# 7.6. MacPorts Membership

## Requirements for Becoming a MacPorts Committer

To become a MacPorts committer, one must first contribute to the project through various means:

- Contributing new ports
- Fixing bugs in existing ports
- Volunteering as a maintainer of non-maintained ports
- Participating in MacPorts development or user support mailing lists
- Contributing documentation

## Application Process

To apply for commit rights, send an email to `macports-mgr@lists.macports.org` with the subject "Commit access: [Your Name]" including:

1. **A description of your application and contributions**
   - Provide evidence of contributions
   - Include direct links to Trac tickets or searches to aid review

2. **Your GitHub username**

3. **A real email address for your MacPorts alias forwarding**

### Email Details

- **Subject:** "Commit access: [Your Name]"
- **Recipient:** `macports-mgr@lists.macports.org`

The PortMgr team will review applications and respond accordingly.
