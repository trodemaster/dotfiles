# Chapter 8. MacPorts Guide Glossary

## Glossary

This section defines a number of words which may be new to the reader, defined in the context of MacPorts.

### MacPorts Guide Terms

The glossary includes definitions for various technical terms related to MacPorts, such as:

- **MacPorts**: A system for compiling, installing, and managing free and open source software, comprised of an infrastructure called MacPorts base and a collection of ports.

- **checksum**: A small piece of data derived from an original that can be used to ensure two files are identical.

Some key terms defined in the glossary include:

- activate phase
- build phase
- configure phase
- destroot phase
- extract phase
- fetch phase
- patch phase

Other notable terms:
- API
- dependency
- PortGroup
- Portfile
- registry
- variant

Technical terms from software development and Unix environments are also included, such as:
- automake
- autoconf
- Tcl
- Subversion
- Xcode Tools
- X11

The glossary serves as a reference for understanding MacPorts-specific terminology and concepts related to software management and development.
